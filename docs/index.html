Equations¶

pyexlatex.models.equation.Equation([eq, …])

Pass sympy or string equations to have them rendered in LaTeX.

Text¶

pyexlatex.models.section.paragraphs.Paragraph(…)

A paragraph, the second smallest section block.

pyexlatex.models.section.paragraphs.SubParagraph(…)

Part of a paragraph, the smallest section block.

pyexlatex.models.section.sections.Section(…)

A section of the document, the largest section type.

pyexlatex.models.section.sections.SubSection(…)

A part of a section, the second largest section type.

pyexlatex.models.section.sections.SubSubSection(…)

A part of a subsection, the third largest section type.

pyexlatex.models.lists.unordered.UnorderedList(items)

Bullet point items.

pyexlatex.models.lists.unordered.UnorderedList(items)

Bullet point items.

pyexlatex.models.format.raw.Raw(contents)

Don’t replace latex characters in this blocks such as or %, useful for adding latex commands or comments manually.

pyexlatex.models.format.text.Text

pyexlatex.models.ref.Ref(contents)

Pass a label to create an in text reference to that label.

pyexlatex.models.footnote.Footnote(contents)

Creates a footnote at the bottom of the page, with a mark next to where the object itself is placed to reference the footnote.

pyexlatex.models.references.citations.Cite(…)

Basic citation command, creates a reference to a document included in the bibliography.

pyexlatex.models.references.citations.CiteP(…)

Paragraph citation command, creates a paragraph (with parentheses) reference to a document included in the bibliography.

pyexlatex.models.references.citations.CiteT(…)

In text citation command, creates an in text (without parentheses) reference to a document included in the bibliography.
